Dúvidas sobre o MSTMC

Página dedicada a responder dúvidas comuns sobre a proposta, o funcionamento e o objetivo do projeto.

O que é o MSTMC?

É uma solução conceitual da Aeris Mars para controle de tráfego, clima e emergências em Marte.

O sistema usa dados reais?

Não. Este protótipo usa dados simulados para representar uma interface funcional no contexto acadêmico.

Qual é o objetivo do site?

Apresentar o projeto, seus módulos principais e demonstrar conceitos de HTML, CSS e JavaScript.

Por que Marte?

Porque o tema da Global Solution envolve economia espacial, exploração e soluções tecnológicas inovadoras.
