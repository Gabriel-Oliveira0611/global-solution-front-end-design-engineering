# Aeris Mars - MSTMC

Projeto acadêmico desenvolvido para a Global Solution 2026/1 da FIAP.

## Descrição

O site apresenta a solução Mars Spacial Traffic & Mission Control, uma plataforma conceitual da Aeris Mars para monitoramento de tráfego aéreo marciano, clima e emergências.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub
